## Stories

export const translations = addTranslations({
    'ui-date.day.placeholder': 'Day',
    'ui-date.month.placeholder': 'Month',
    'ui-date.year.placeholder': 'Year',
    'ui-date.date.error': 'You must be at least {minimumAge} to sign up.',
    'ui-date.date.invalid': 'Invalid date',
});

The date field is form control element. It's used to take a date value from the user.

When the date is invalid the field will emit a change event with an empty string.

<Preview>
    <Story name="Default">
        <magic-ui-date />
    </Story>
</Preview>

```html
<magic-ui-date />
```

### Minimum Age

As age verification is a common use case for the date field the `minimum-age` attribute can be used to specify the minimum age allowed.

<Preview>
    <Story name="Minimum Age">
        <magic-ui-date value="2000-09-02" minimum-age="30" />
    </Story>
</Preview>

```html
<magic-ui-date minimum-age="30" />
```

### Value

A value can be pre-specified using the `value` attribute. The format is `YYYY-MM-DD`.

<Preview>
    <Story name="With Value">
        <magic-ui-date value="1990-12-30" />
    </Story>
</Preview>

```html
<magic-ui-date value="1990-12-30" />
```

### Info

An informational message can be provided attached to the field with the `info` property.

<Preview>
    <Story name="With Info">
        <magic-ui-date info="Enter the issue date exactly as it's specified on your passport." />
    </Story>
</Preview>

```html
<magic-ui-date info="Enter the issue date exactly as it's specified on your passport." />
```
