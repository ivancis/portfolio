## Stories

The ui-link element is a simple wrapper for an anchor tag. Any attributes defined on it are passed
down to the underlying `a` tag.

<Preview>
    <Story name="Default">
        <magic-ui-link href="https://google.com" target="_blank">
            My long link
        </magic-ui-link>
    </Story>
</Preview>

```html
<magic-ui-link href="https://google.com" target="_blank">
    My long link
</magic-ui-link>
```

### Disabled Link

A link can be disabled by adding the `disabled` attribute.

Any click events will have the default action prevented.

<Preview>
    <Story name="Disabled">
        <magic-ui-link disabled href="https://google.com" target="_blank">
            My long link
        </magic-ui-link>
    </Story>
</Preview>

```html
<magic-ui-link disabled href="https://google.com" target="_blank">
    My long link
</magic-ui-link>
```
