## Stories

By default form field renders an input field

<Preview isExpanded language="markup">
    <Story name="Default" inline>
        <magic-ui-form-field />
    </Story>
</Preview>

```html
<magic-ui-form-field />
```

### With label

When you add a label it will be rendered above the input

<Preview isExpanded language="markup">
    <Story name="Label" inline>
        <magic-ui-form-field label="Foo" />
    </Story>
</Preview>

```html
<magic-ui-form-field label="Foo" />
```

### With state

You can add various states to the form field

<Preview isExpanded language="markup">
    <Story name="States" inline>
        <div>
            <magic-ui-form-field has-error />
            <magic-ui-form-field has-success />
            <magic-ui-form-field has-warning />
        </div>
    </Story>
</Preview>

```html
<magic-ui-form-field has-error />
<magic-ui-form-field has-success />
<magic-ui-form-field has-warning />
```

### With error message

You can also provide a message to the error state

<Preview isExpanded language="markup">
    <Story name="Message" inline>
        <magic-ui-form-field message="Provided value is invalid" has-error />
    </Story>
</Preview>

```html
<magic-ui-form-field
    message="Provided value is invalid"
    has-error
/>
```

### No icon

You can remove icon from error and success states

<Preview isExpanded language="markup">
    <Story name="No icon" inline>
        <div>
            <magic-ui-form-field has-error no-icon />
            <magic-ui-form-field has-success no-icon />
        </div>
    </Story>
</Preview>

```html
<magic-ui-form-field has-error no-icon />
<magic-ui-form-field has-success no-icon />
```

### With info

You can add and information to the field, e. g. describing what's the correct
format of the field

<Preview isExpanded language="markup">
    <Story name="Info" inline>
        <magic-ui-form-field
            label="Date of birth"
            info="Please enter a date in the following format: YYYY/MM/DD"
            has-info
        />
    </Story>
</Preview>

```html
<magic-ui-form-field
    label="Date of birth"
    info="Please enter a date in the following format: YYYY/MM/DD"
    has-info
/>
```

### Margin

By default form field has bottom margin set. You can disable that
with `no-margin` option

<Preview isExpanded language="markup">
    <Story name="Margin" inline>
        <div>
            <magic-ui-form-field no-margin />
            <magic-ui-form-field />
        </div>
    </Story>
</Preview>

```html
<magic-ui-form-field no-margin />
<magic-ui-form-field />
```

### Centered

Use `centered` option to center align form field content

<Preview isExpanded language="markup">
    <Story name="Centered" inline>
        <magic-ui-form-field label="Foo" placeholder="Bar" centered />
    </Story>
</Preview>

```html
<magic-ui-form-field
    label="Foo"
    placeholder="Bar"
    centered
/>
```

### Custom field

Form field can be used as a container for any field

<Preview isExpanded language="markup">
    <Story name="Custom" inline>
        <magic-ui-form-field label="Foo">
            <magic-ui-date />
        </magic-ui-form-field>
    </Story>
</Preview>

```html
<magic-ui-form-field label="Foo">
    <magic-ui-date />
</magic-ui-form-field>
```
