## Stories

The peek-password component is used for when you want to allow the user to show/hide the
password they are entering.

<Preview>
    <Story name="Default">
        <magic-ui-peek-password value="My password" />
    </Story>
</Preview>

```html
<magic-ui-peek-password />
```

### Shown

The `shown` attribute is for when you want to control the shown status from outside
of the component.

<Preview>
    <Story name="Shown">
        <magic-ui-peek-password value="My password" shown />
    </Story>
</Preview>

```html
<magic-ui-peek-password shown />
```
