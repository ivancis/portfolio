### Stories

The radio element is a form control element. It displays a radio button.

<Preview>
    <Story name="Default">
        <magic-ui-radio />
    </Story>
</Preview>

```html
<magic-ui-radio />
```

### Checked state

The checked state can be controled via the `checked` attribute.

<Preview>
    <Story name="Checked">
        <magic-ui-radio checked />
    </Story>
</Preview>

```html
<magic-ui-radio checked />
```

### Adding Labels

The label can be passed through as the default slot.

<Preview>
    <Story name="Label">
        <div>
            <magic-ui-radio>My radio with label 1</magic-ui-radio>
            <magic-ui-radio>My radio with label 2</magic-ui-radio>
        </div>
    </Story>
</Preview>

```html
<magic-ui-radio>My radio with label 1</magic-ui-radio>
<magic-ui-radio>My radio with label 2</magic-ui-radio>
```

### Inline Radios

When using a label you can specify the radio element should be inline styled.

The containing element should have the appropriate styles to handle this.

<Preview>
    <Story name="Inline">
        <div>
            <magic-ui-radio inline>My inline radio with label 1</magic-ui-radio>
            <magic-ui-radio inline>My inline radio with label 2</magic-ui-radio>
        </div>
    </Story>
</Preview>

```html
<magic-ui-radio inline>My inline radio with label 1</magic-ui-radio>
<magic-ui-radio inline>My inline radio with label 2</magic-ui-radio>
```

### Bold Label

To make the label bold you can use the `bold` attribute.

<Preview>
    <Story name="Bold">
        <magic-ui-radio bold>My radio with bold label</magic-ui-radio>
    </Story>
</Preview>

```html
<magic-ui-radio bold>My radio with bold label</magic-ui-radio>
```
