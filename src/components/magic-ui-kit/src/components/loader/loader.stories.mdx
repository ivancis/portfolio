## Stories

export const container = {
    position: 'relative',
    minHeight: '200px',
};

The `magic-ui-loader` element displays a loader. It positions itself in the center of the container.

<Preview>
    <Story name="Default">
        <magic-ui-loader />
    </Story>
</Preview>

```html
<magic-ui-loader />
```

### In Container

To move the loader around, the container should have the appropriate styles applied, for example
in this case the container has a height of `200px` and `relative` positioning.

<Preview>
    <Story name="In Container">
        <div style={container}>
            <magic-ui-loader />
        </div>
    </Story>
</Preview>

```html
<div class="container">
    <magic-ui-loader />
</div>

<style>
    .container {
        position: relative;
        min-height: 200px;
    }
</style>
```

### Changing the color

To modify the colour of the loader use the `--magic-loader-stroke` css variable. The variable accepts any standard CSS color.

<Preview>
    <Story name="With Color">
        <magic-ui-loader style={{ '--magic-loader-stroke': 'red' }} />
    </Story>
</Preview>

```html
<magic-ui-loader style="--magic-loader-stroke: red" />
```
