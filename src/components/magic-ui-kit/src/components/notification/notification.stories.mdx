## Stories

Toast style notifications can be made with the `magic-ui-notification` element.

Using the `variant` attribute you can control how the notification looks.

<Preview>
    <Story name="Variants">
        <div>
            <magic-ui-notification
                variant="message"
                content="My message notification"
            />
            <magic-ui-notification
                variant="info"
                content="My info notification"
            />
            <magic-ui-notification
                variant="success"
                content="My success notification"
            />
            <magic-ui-notification
                variant="alert"
                content="My alert notification"
            />
            <magic-ui-notification
                variant="error"
                content="My error notification"
            />
        </div>
    </Story>
</Preview>

```html
<magic-ui-notification variant="message" content="My message notification" />
<magic-ui-notification variant="info" content="My info notification" />
<magic-ui-notification variant="success" content="My success notification" />
<magic-ui-notification variant="alert" content="My alert notification" />
<magic-ui-notification variant="error" content="My error notification" />
```

### Dismissible

The `dismissible` attribute specifies whether to include a close icon in the top corner.

Once clicked a `close` event is emitted from the element.

<Preview>
    <Story name="Dismissible">
        <div>
            <magic-ui-notification
                dismissible
                variant="info"
                content="My info notification"
            />
            <magic-ui-notification
                dismissible
                variant="success"
                content="My success notification"
            />
            <magic-ui-notification
                dismissible
                variant="alert"
                content="My alert notification"
            />
            <magic-ui-notification
                dismissible
                variant="error"
                content="My error notification"
            />
        </div>
    </Story>
</Preview>

```html
<magic-ui-notification dismissible variant="info" content="My info notification" />
<magic-ui-notification dismissible variant="success" content="My success notification" />
<magic-ui-notification dismissible variant="alert" content="My alert notification" />
<magic-ui-notification dismissible variant="error" content="My error notification" />
```
