## Stories

Tags simply take the slotted content and wrap them in a pill container.

<Preview>
    <Story name="Default">
        <magic-ui-tag>NEW GAME</magic-ui-tag>
    </Story>
</Preview>

```html
<magic-ui-tag>NEW GAME</magic-ui-tag>
```

### Inline

Displaying `inline-block` will make the pill take the width of it's content.

<Preview>
    <Story name="Inline">
        <div>
            <magic-ui-tag style={{ display: 'inline-block' }}>
                NEW GAME
            </magic-ui-tag>
            <magic-ui-tag style={{ display: 'inline-block' }}>
                SLOTS
            </magic-ui-tag>
            <magic-ui-tag style={{ display: 'inline-block' }}>
                BLAH
            </magic-ui-tag>
        </div>
    </Story>
</Preview>

```html
<magic-ui-tag style="display: inline-block;">NEW GAME</magic-ui-tag>
```
