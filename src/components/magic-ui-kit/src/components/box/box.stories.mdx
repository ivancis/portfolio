## Stories

Box is a super simple container element, by default it has no shadow.

<Preview>
    <Story name="With Content">
        <magic-ui-box>{content}</magic-ui-box>
    </Story>
</Preview>

### Elevation

The box's elevation attribute can be used to visually raise the box out from the page:

<Preview>
    <Story name="With Elevations">
        <div>
            <magic-ui-box add-bottom-margin elevation="1">
                Elevation 1: {content}
            </magic-ui-box>
            <magic-ui-box add-bottom-margin elevation="2">
                Elevation 2: {content}
            </magic-ui-box>
            <magic-ui-box add-bottom-margin elevation="3">
                Elevation 3: {content}
            </magic-ui-box>
            <magic-ui-box add-bottom-margin elevation="4">
                Elevation 4: {content}
            </magic-ui-box>
            <magic-ui-box add-bottom-margin elevation="5">
                Elevation 5: {content}
            </magic-ui-box>
        </div>
    </Story>
</Preview>

```html
<magic-ui-box elevation="1">Box Content</magic-ui-box>
```

### Disable Padding

You can disable the padding in the bottom of the content by providing the `no-bottom-padding` attribute.

<Preview>
    <Story name="Without Bottom Padding">
        <magic-ui-box no-bottom-padding elevation="3">
            {content}
        </magic-ui-box>
    </Story>
</Preview>

```html
<magic-ui-box no-bottom-padding elevation="3">Box Content</magic-ui-box>
```
