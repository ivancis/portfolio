## Stories

Step progress bar shows given number of steps with current step selected

<Preview>
    <Story name="Default">
        <magic-ui-step-progress-bar steps="5" />
    </Story>
</Preview>

```html
<magic-ui-step-progress-bar steps="5" />
```

### Current step

To mark a step as current use `current-step` attribute.
All the previous steps will be marked as done.

<Preview>
    <Story name="Current Step">
        <magic-ui-step-progress-bar steps="5" current-step="3" />
    </Story>
</Preview>

```html
<magic-ui-step-progress-bar steps="5" current-step="3" />
```
