## Stories

Code entry is a form control element, it's used to collect numeric inputs of a specified length. By default it'l show 4 number inputs and moves to the next field automatically as they're filled.

<Preview>
    <Story name="Default">
        <magic-ui-code-entry />
    </Story>
</Preview>

```html
<magic-ui-code-entry />
```

### Pre-filled value

The value of the field can be pre-filled by providing the `value` attribute.

<Preview>
    <Story name="With Value">
        <magic-ui-code-entry value="123" />
    </Story>
</Preview>

```html
<magic-ui-code-entry value="123" />
```

### Length

The length of the code is specified by the `length` attribute.

<Preview>
    <Story name="With Length">
        <magic-ui-code-entry length="12" />
    </Story>
</Preview>

```html
<magic-ui-code-entry length="12" />
```
