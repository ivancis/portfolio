## Stories

Badges accept a number up to 99, for example we can use it to notify to the user they have one new message.

<Preview>
    <Story name="With Text">
        <magic-ui-badge>{text('Text', '1')}</magic-ui-badge>
    </Story>
</Preview>

```html
<magic-ui-badge>1</magic-ui-badge>
```

### With Element

Of course since it's just HTML we can accept any html element inside for example a strong tag:

<Preview>
    <Story name="With Element">
        <magic-ui-badge>
            <strong>12</strong>
        </magic-ui-badge>
    </Story>
</Preview>

```html
<magic-ui-badge><strong>12</strong></magic-ui-badge>
```

### Floating

Sometimes you want the badge to float to the top right of the container, to do so just specify the floats property.

<Preview>
    <Story name="Floating">
        <magic-ui-badge floats={boolean('Floats?', true)}>40</magic-ui-badge>
    </Story>
</Preview>

```html
<magic-ui-badge floats>40</magic-ui-badge>
```

### Sizing

The size of the badge can be controlled with the `--magic-badge-size` css variable.

<Preview>
    <Story name="Badge Size">
        <magic-ui-badge style={{ '--magic-badge-size': text('Size', '32px') }}>
            99+
        </magic-ui-badge>
    </Story>
</Preview>

```html
<magic-ui-badge style="--magic-badge-size: 32px;">40</magic-ui-badge>
```
