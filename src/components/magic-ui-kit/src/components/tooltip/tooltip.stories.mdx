## Stories

export const dark = {
    backgroundColor: '#333',
    padding: '16px',
    color: '#fff',
    borderRadius: '5px',
};

The tooltip component provides a method of opening, you guessed it, tooltips.
The element accepts any child element that when hovered over will display a popover with the content required.

The text to show is contained within the `label` attribute.

<Preview>
    <Story name="Default">
        <div style={dark}>
            <magic-ui-tooltip
                style={{ display: 'inline-block' }}
                label="Text to show"
            >
                Hover Over Me!
            </magic-ui-tooltip>
        </div>
    </Story>
</Preview>

```html
<magic-ui-tooltip style="display: inline-block" label="Text to show">
    Hover Over Me!
</magic-ui-tooltip>
```

Note: You must set the display to `inline-block` if the clickable element has no explicit width.
