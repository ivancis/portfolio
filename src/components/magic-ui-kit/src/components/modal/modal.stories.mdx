## Stories

export const styles = {
    header: {
        fontSize: '16px',
        margin: 'auto',

Modal shows content in a modal dialog

<Preview>
    <Story name="Default">
        <magic-ui-modal>
            <p>Lorem ipsum dolor sit amet</p>
        </magic-ui-modal>
    </Story>
</Preview>

```html
<magic-ui-modal><p>Lorem ipsum dolor sit amet</p></magic-ui-modal>
```

### Header

Use `header` slot to add header to the dialog

<Preview>
    <Story name="Header">
        <magic-ui-modal>
            <h3 slot="header" style={styles.header}>
                Lorem ipsum
            </h3>
            <p>Lorem ipsum dolor sit amet</p>
        </magic-ui-modal>
    </Story>
</Preview>

```html
<magic-ui-modal>
    <h3 slot="header">Lorem ipsum</h3>
    <p>Lorem ipsum dolor sit amet</p>
</magic-ui-modal>
```

## Size

Modal dialog can have different sizes

<Preview>
    <Story name="Size">
        <magic-ui-modal size="small">

        </magic-ui-modal>

```html
<magic-ui-modal size="small"><p>Lorem ipsum dolor sit amet</p></magic-ui-modal>
```

## Action buttons

Modal has two action buttons available: close and go back

<Preview>
    <Story name="Close">
        <magic-ui-modal dismissible>
            <p>Lorem ipsum dolor sit amet</p>
        </magic-ui-modal>
    </Story>
</Preview>

<Preview>
    <Story name="Go Back">
        <magic-ui-modal back-arrow dismissible>
            <p>Lorem ipsum dolor sit amet</p>
        </magic-ui-modal>
    </Story>
</Preview>

```html
<magic-ui-modal dismissible><p>Lorem ipsum dolor sit amet</p></magic-ui-modal>

<magic-ui-modal back-arrow dismissible>
    <p>Lorem ipsum dolor sit amet</p>
</magic-ui-modal>
```
