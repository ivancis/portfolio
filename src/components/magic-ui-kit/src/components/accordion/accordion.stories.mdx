### Initially Visible

The accoordion can be opened by default using the `initiallyVisible` prop.

<Preview>
    <Story name="Initially Visible">
        <magic-ui-accordion initially-visible={boolean('Is visible?', true)}>
            <b slot="toggle">{text('Toggle Text', 'Default Toggle Text')}</b>
            <ul slot="content">
                <p>This is some content to display!</p>
                <p>It's toggled every time the page is toggled</p>
            </ul>
        </magic-ui-accordion>
    </Story>
</Preview>

```html
<magic-ui-accordion initially-visible>
    <b slot="toggle">Toggle Text</b>
    <ul slot="content">
        <p>This is some content to display!</p>
        <p>It's toggled every time the page is toggled</p>
    </ul>
</magic-ui-accordion>
```

### Full Width

You can force the toggle take all of the available space by providing the `wide` property.

<Preview>
    <Story name="Full Width">
        <magic-ui-accordion wide={boolean('Is wide?', true)}>
            <b slot="toggle">{text('Toggle Text', 'Default Toggle Text')}</b>
            <ul slot="content">
                <p>This is some content to display!</p>
                <p>It's toggled every time the page is toggled</p>
            </ul>
        </magic-ui-accordion>
    </Story>
</Preview>

```html
<magic-ui-accordion wide>

        <p>It's toggled every time the page is toggled</p>
```
