## Stories

export const style = {
    '--magic-progress-bar-track-background-transparent': 'rgba(0, 0, 0, .05)',
};

By default progress of the bar is set to 0

<Preview isExpanded language="markup">
    <Story name="Default" inline>
        <magic-ui-progress-bar style={style} />
    </Story>
</Preview>

```html
<magic-ui-progress-bar />
```

### With progress

When progress is set it will be reflected on the bar

<Preview>
    <Story name="Progress">
        <magic-ui-progress-bar progress="25" style={style} />
    </Story>
</Preview>

```html
<magic-ui-progress-bar progress="25" />
```

### With full progress

When progress is set to 100 an effect will be added to the bar

<Preview>
    <Story name="Full">
        <magic-ui-progress-bar progress="100" style={style} />
    </Story>
</Preview>

```html
<magic-ui-progress-bar progress="100" />
```

### Animated

By default the progress bar animates between progress values to modify the duration of animation you can use the `--magic-duration` CSS variable

<Preview>
    <Story name="Animated">
        <div>
            <style>{`
                @keyframes progress-animation {
                    0% {
                        --magic-progress-bar-width: 0%;
                    }
                    50% {
                        --magic-progress-bar-width: 100%;
                    }
                    100% {
                        --magic-progress-bar-width: 0%;
                    }
                }
            `}</style>
            <magic-ui-progress-bar
                style={{
                    ...style,
                    '--magic-duration': text('Animation Duration', '1s'),
                    animation:
                        'progress-animation 2s cubic-bezier(1, 0, 1, 0) infinite',
                }}
            />
        </div>
    </Story>
</Preview>
