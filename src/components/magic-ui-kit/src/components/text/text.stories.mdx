## Stories

The text component is usually used to display some text from the CMS.
By default it's not doing much and just taking the content as normal text from the `content` property and applying
some appropriate styles.

### Markdown

By providing the `is-markdown` attribute the element will transform the text parsing any markdown passed to it.

In markdown mode, new lines are preserved as you would expect.

<Preview>
    <Story name="With Markdown">
        <magic-ui-text
            is-markdown
            content="# I am a heading defined in markdown!"
        />
    </Story>
</Preview>

```html
<magic-ui-text is-markdown content="# I am a heading defined in markdown!" />
```

### Param Interpolation

With the `params` property, you can pass through an object as a JS property to interpolate values.

Anything found within `{}` brackets will be interpolated. If no interpolation param is found it'l not transform the text.

<Preview>
    <Story name="With Params">
        <ComponentInterop props={{ params: { hello: 'World' } }}>
            <magic-ui-text
                is-markdown
                content="Hello {hello}! (View the source)"
            />
        </ComponentInterop>
    </Story>
</Preview>

```html
<magic-ui-text id="text" is-markdown content="Hello {hello}!" />

<script>
    const text = document.getElementById('text');
    text.params = { hello: 'World' };
</script>
```
