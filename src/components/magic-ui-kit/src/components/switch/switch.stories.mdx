## Stories

Switch renders a switchable control similar to checkbox

<Preview>
    <Story name="Default">
        <magic-ui-switch />
    </Story>
</Preview>

```html
<magic-ui-switch />
```

### Label

Use slot to add label to the switch

<Preview>
    <Story name="Label">
        <magic-ui-switch id="foo">Foo</magic-ui-switch>
    </Story>
</Preview>

```html
<magic-ui-switch id="foo">Foo</magic-ui-switch>
```

### Right

There is a possibility to put switch after the label

<Preview>
    <Story name="Right">
        <div style={{ width: '100px' }}>
            <magic-ui-switch id="foo" right>
                Foo
            </magic-ui-switch>
        </div>
    </Story>
</Preview>

```html
<div style="width: 100px;">
    <magic-ui-switch id="foo" right>Foo</magic-ui-switch>
</div>
```
