### Inline Buttons

By default buttons are inline however they do not add spacing between each button/textual content.
You can use the `add-space` attribute in order to add a sufficient gutter between elements:

<Preview>
    <Story name="With Added Space">
        <div>
            <magic-ui-button variant="primary" add-space>
                OK
            </magic-ui-button>
            <magic-ui-button variant="error" add-space>
                CANCEL
            </magic-ui-button>
        </div>
    </Story>
</Preview>

```html
<div>
    <magic-ui-button variant="primary" add-space>OK</magic-ui-button>
    <magic-ui-button variant="error" add-space>CANCEL</magic-ui-button>
</div>
```

### Usage with Icons

When using an icon with a button you should specify the `icon` attribute.

<Preview>
    <Story name="With Icon">
        <magic-ui-button icon variant="error">
            <magic-ui-icon style={{ width: 24, height: 24 }} name="denied" />
        </magic-ui-button>
    </Story>
</Preview>

```html
<magic-ui-button icon variant="error">
    <magic-ui-icon name="denied"></magic-ui-icon>
</magic-ui-button>
```

To make the button round with icon, specify the `round` attribute.

<Preview>
    <Story name="Round With Icon">
        <magic-ui-button icon round variant="error">
            <magic-ui-icon style={{ width: 24, height: 24 }} name="denied" />
        </magic-ui-button>
    </Story>
</Preview>

```html
<magic-ui-button icon round variant="error">
    <magic-ui-icon name="denied"></magic-ui-icon>
</magic-ui-button>
```

To make the button with right aligned icon and text, specify the `icon right` attributes and add content as tag with `class="button-text"`. Tags order matters.

<Preview>
    <Story name="Right Aligned Icon">
        <magic-ui-button icon right variant="error">
            <span class="button-text">Button icon right</span>
            <magic-ui-icon
                style={{ width: 24, height: 24 }}
                name="close-solid"
            />
        </magic-ui-button>
    </Story>
</Preview>

```html
<magic-ui-button icon right variant="error">
    <span class="button-text">Button icon right</span>
    <magic-ui-icon name="close-solid"></magic-ui-icon>
</magic-ui-button>
```

To make the button with left aligned icon and text, specify the `icon left` attributes and add content as tag with `class="button-text"`. Tags order matters.

<Preview>
    <Story name="Left Aligned Icon">
        <magic-ui-button icon left variant="success">
            <magic-ui-icon
                style={{ width: 24, height: 24 }}
                name="checked-solid"
            />
            <span class="button-text">Button icon left</span>
        </magic-ui-button>
    </Story>
</Preview>

```html
<magic-ui-button icon left variant="error">
    <magic-ui-icon name="solid-checked"></magic-ui-icon>
    <span class="button-text">Button icon left</span>
</magic-ui-button>
```

### Links as buttons

Using the `button-tag` attribute you can change the rendered element, for example you could use it to create an `a` tag and specify the `href` attribute as any unrecognized attributes are passed to the child element.

<Preview>
    <Story name="Button Tag">
        <magic-ui-button
            button-tag="a"
            href="https://google.com"
            target="_blank"
            variant="primary"
        >
            Go to Google
        </magic-ui-button>
    </Story>
</Preview>

```html
<magic-ui-button
    button-tag="a"
    href="https://google.com"
    target="_blank"
    variant="primary"
>
    Go to Google
</magic-ui-button>
```
